---
description: Analyze WhatsApp Business Message Delivery Reports (MDRs) to find where a sales funnel is leaking
---

Invoke the sent-skills:messaging-performance-analyzer skill.

Begin by pinning the question. Ask:
1. What's the specific concern? ("delivery dropped today", "this template underperforms in Brazil", "leads aren't replying")
2. The cohort: which template(s), which country/countries, which tenant(s), which time window?
3. The data source: CSV/JSON dump, paste, log file path, or a database the user can query?

If the question is too vague to answer (e.g. "how are we doing"), narrow it before pulling data. Always require a defined cohort.
